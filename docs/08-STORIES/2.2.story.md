# STORY-2.2: DB Cleanup — Migration de Schema

## Contexto

O banco tem tabelas e funções legadas/duplicadas identificadas durante auditoria de 2026-06-24:
- `members` (tabela legada, substituted por `profiles`)
- `consortium_groups` (deprecada em 2026-02-10, substituída por `groups`)
- `is_admin()` — 3 versões duplicadas da mesma função
- `leads` (se vazia, consolidar em `prospects`)

A migration `20260624000001_cleanup_legacy_schema.sql` foi preparada com todas as salvaguardas necessárias.
